# 🚀 CRM Project (MERN Stack)

A full-stack CRM application built using **React (Vite)**, **Node.js**, **Express**, and **MongoDB**.

### Frontend
- React (Vite)

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
